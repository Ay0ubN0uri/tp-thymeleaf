# How to install

```bash
git clone https://github.com/Ay0ubN0uri/tp-thymeleaf.git
cd tp-thymeleaf
mvn clean install
npm install
npm run build
mvn spring-boot:run
```

# Using docker

```bash
git clone https://github.com/Ay0ubN0uri/tp-thymeleaf.git
cd tp-thymeleaf
docker-compose up
```
